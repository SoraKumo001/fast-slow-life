# Fast Slow Life

村を発展させながらダンジョンを攻略していく、テキストベースの街経営＆ダンジョンRPGシミュレーションゲームです。
本リポジトリは、街経営＆ダンジョンRPGシミュレーションゲーム「Fast Slow Life」のゲームデザイン原案（仕様書・パラメータ設計）および、Web（React + TypeScript）での実装コードを格納しています。

## 💡 コアシステム

### 1. 施設での素材加工とアップグレード

村の施設は単なる売買や強化の場所ではありません。
採取してきた一次素材（食料、鉱石、薬草など）を施設で加工し、より価値の高い中間素材や装備品、薬品を「クラフト」できます。
また、集めた素材を消費して施設自体の機能を「アップグレード」することで、新たなレシピの解放や生産効率の強化が行えます。

### 2. 職業適性に応じた村人の自動行動

プレイヤーは村人一人ひとりに対し、「採取」か「討伐」かの大まかな方針（指示）を出します。
村人は与えられた指示に対し、自身の職業（農民、鉱夫、猟師、戦士など）の適性やステータスに基づいて、自動的に「最も適した具体的な行動（どの素材を採取するか、どの敵を討伐するか）」を決定して実行します。

### 3. 調理と食料Buffシステム

収穫した小麦・野菜・生肉や、討伐で得た魔獣の肉を調理場で加工し、村人のステータスを一時的に強化する「料理」を作ることができます。
村人は毎時間、倉庫から自動的に食料を消費しますが、調理済みの料理を食べている間はSTR・INT・VIT・DEX・AGI・最大HP・最大スタミナにボーナスが適用されます。

### 4. 交易・経済システム

交易所と交易馬車を使い、3つの街（コモレビ村・港町アイアンポート・魔法都市マギカ）との交易が可能です。
アイテムの輸出でゴールドと友好度を得られ、友好度が上がるとより良い品物を仕入れられるようになります。投資で交易時間の短縮も可能です。
毎日変動するマーケットトレンドを活用し、高値でアイテムを売り捌くことも重要です。

### 5. 資源生産施設

農場・伐採所・採石場は、時間経過で自動的に素材を生産します。施設レベルを上げることで生産量が向上します。

### 6. ダンジョン脅威度と目標ボス

ゲーム開始時、プレイヤーには「討伐すべきボス」が提示されます。
未攻略ダンジョンの**脅威度（Threat Level）**は時間経過と共に**加速度的に上昇**し、100% に達した時点で**即ゲームオーバー**となります。プレイヤーは**ゴールドによるお布施**で脅威度を下げられますが、コストは軽減量に応じて指数逓増するため、長期的解決はボス討伐が最善となります。
ボス討伐に成功すると**全ダンジョンの脅威度がリセット**され、ゲームは終了せずさらに手強い「次の目標ボス」が設定されて継続します（**脅威度が 100% に達した時点で即ゲームオーバー**となり、それまでの発展度に応じた「ソウルポイント」を獲得できます。これを使用して永続的なバフを獲得し、次の周回（強くてニューゲーム）へと挑むことができます）。
詳細は [`docs/system_design.md` §11](./docs/system_design.md#11-ダンジョン脅威度システム) および [`docs/dungeons_and_combat.md` §1.5](./docs/dungeons_and_combat.md#15-脅威度とお布施システム) を参照。

### 実装外の要素（今後の拡張予定）

- **装備の強化（+1, +2, ...）システム**: 現在は装備の作成のみ。強化は未実装。
- **大成功による装備性能向上**: 現在は大成功時は成果物2倍のみで、性能向上は未実装。
- **食料価格の個別設定**: 各村人の購買力に応じた価格判定は実装済みだが、完全な市場経済シミュレーションは未実装。

## 🛠️ 基本仕様（概要）

- **グラフィック**：基本的にテキストおよびシンプルなUIベース。
- **ゲームサイクル**：リアルタイム時間経過（1時間ステップ進行、オート進行および一時停止）。
- **主要リソース**：
  - ゴールド（資金）
  - 食料（1時間ごとに村人の人数に応じて消費。枯渇すると飢餓デバフが発生）
  - ソウルポイント（周回用資源）
