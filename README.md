# Fast Slow Life

## 概要

- 村を発展させダンジョンを攻略するゲーム
- 基本的にテキストベースでキャラクターの画像などは出さない
- 村人は村の中で生産活動をするか、村の外で採取や討伐の活動をする
- 村の外にはフィールドやダンジョンがあり、探索によって行ける範囲が広がる
- 新しいフィールドやダンジョンに行くにはボスを倒す必要がある
- 一定期間経過までに所定のボスを倒さないとゲームオーバー
- ゲームオーバーになった場合は、その時点の所定のボーナスポイントを使って有利な状態で再スタートできる

## ゲームの進行

- 村人は採取や生産によって必要なアイテムを生産できる
- アイテムは村の施設や装備の強化に利用でき、交易で売ることも出来る
- 村人は作業や戦闘によって成長し能力やスキルを獲得する
- 村の施設を建築することで村人の作業効率や生産量を向上させたり、新しい作業や生産を可能にする
- 一定のコストを払うことで村人を転職させることができ、一度転職した職業にはコスト無しで切り替えられる

## フィールドやダンジョン

- フィールドやダンジョンには素材があり、採取を行うことができる
- フィールドやダンジョンにはモンスターがおり、討伐を行うことができる
- フィールドやダンジョンには探索率が設定されており、探索を進めると上昇する
- 探索率が上がると、新しい素材やモンスターを発見する
- フィールドの探索率がある程度上昇すると、フィールドのボスを討伐できるようになる
- ボスを討伐すると、新しいフィールドやダンジョンに行けるようになる
