# ダンジョン＆戦闘設計書：エリアと戦闘ロジック

ゲームに登場するフィールド・ダンジョンのデータ設計、探索率の進行仕様、および戦闘におけるダメージ計算や報酬決定ロジックの設計です。

---

## 1. フィールド・ダンジョン（エリア）設計

プレイヤーは村人を以下のエリアに派遣して、探索・討伐・採取を行います。
各エリアには「探索度（0% 〜 100%）」があり、探索度の上昇に伴い、よりレアな素材の発見や強力なモンスターとの遭遇が発生します。

### エリア探索率の進行と要素解放

- 毎日、エリアに派遣された村人のステータスに応じて探索度（%）が上昇します。
- **1日あたりの探索度上昇値**：
  $$\text{探索上昇量(\%)} = \frac{\text{村人のDEX} \times 0.2 + \text{村人のAGI} \times 0.2}{\text{エリアの探索難易度}}$$
- **要素の解放**：
  - エリアの探索度が上がるにつれて、採取可能な素材や出現する魔物の種類が段階的に増えていきます。
  - 解放された素材や魔物はキャラクターに割り当て可能になり、自動または手動指示で回収・討伐を行えるようになります。
- **ボス討伐の条件**：
  - 探索度が 100% になると、そのエリアの「ボス討伐」への挑戦が可能になります。
  - **初討伐のボス**に対しては、自動行動による討伐は行われません。プレイヤーがキャラクターへ**直接討伐を指示した場合のみ**、ボス戦の行動を開始します。

### 採取・討伐の進捗ゲージとリスポーン [NEW]

各素材スポットおよびモンスターは、「進捗ゲージ（0%〜100%）」を個別に持ちます。

- **進捗ゲージの上昇**：村人が同エリアで採取・討伐行動を行うたびに、村人のステータス（採取はDEX、討伐はAGI）と難易度に応じてゲージが上昇します。
  - **採取ゲージ上昇式**：`(DEX * 0.8 + 10) / 採取難易度`
  - **討伐ゲージ上昇式**：`(AGI * 0.8 + 10) / モンスターレベル`
- **アイテム取得・戦闘発生**：進捗ゲージが 100% に達した時点で、採取ならアイテム獲得、討伐なら戦闘が発生します。
- **リスポーン待機**：アイテム獲得または討伐後、スポット/モンスターはリスポーン待機時間に入ります。待機時間（時間単位）は種類によって異なり、待機中は採取・討伐の対象外となります。
  - 一般素材・ゴブリン等：3〜4時間
  - 鉄鉱石・オーク等：6〜8時間
  - 魔法石：12時間
  - ゴーレム・キマイラ等の強敵：16時間
- **UIへの反映**：ダンジョンパネルの各素材・モンスター表示に進捗ゲージが表示され、リスポーン待機中は残り時間が表示されます。

## 2. 戦闘システムとダメージ計算式

戦闘はテキストログ形式の自動ターン制バトルです。1回の戦闘は最大10ターン行われ、敵のHPを0にすれば勝利、村人のHPが0になるかターン超過で敗北（一時撤退）となります。

### ① 命中・回避判定

攻撃が命中するかどうかは、アタッカーの `DEX` とディフェンダーの `AGI` の差で決まります。
$$\text{命中率(\%)} = 85 + (\text{アタッカーのDEX} - \text{ディフェンダーのAGI}) \times 1.5$$
_(最低命中率：50%、最高命中率：100%。1〜100の乱数が命中率以下ならヒット)_

---

### ② ダメージ計算式

アタッカーの攻撃タイプ（物理/魔法）に応じて、以下の式でダメージを算出します。

#### 物理攻撃（戦士、猟師、農民、鉱夫など）

$$\text{与ダメージ} = \max\left(1, (\text{STR} \times 1.5 + \text{武器攻撃力}) - (\text{敵の物理防御力} + \text{敵のVIT} \times 0.5)\right)$$

#### 魔法攻撃（魔術師、僧侶、薬師など）

$$\text{与ダメージ} = \max\left(1, (\text{INT} \times 1.8 + \text{武器魔力補正}) - (\text{敵の魔法防御力} + \text{敵のINT} \times 0.5)\right)$$

- **クリティカルヒット**：
  - `DEX` による判定。確率： $\text{DEX} \times 0.1\%$ （上限 30%）。
  - クリティカル時はダメージが **1.5倍** になり、防御力を 50% 無視します。

---

### ③ 報酬システム (経験値とアイテムドロップ)

戦闘に勝利した場合、村人は「経験値（EXP）」を獲得し、倉庫には「素材アイテム」が追加されます。

#### 獲得経験値

討伐したモンスターの基本EXPと、村人の現在のレベル差によって獲得量が補正されます。
$$\text{獲得EXP} = \text{モンスターの基本EXP} \times \max\left(0.1, 1.0 - (\text{村人のLV} - \text{モンスターのLV}) \times 0.1\right)$$

- レベルが上がりすぎている敵（村人のLVが敵のLVより10以上高い）からは、基本値の 10% しかEXPが得られません。

#### ドロップアイテム判定

各モンスターは固有のドロップテーブルを持ちます。

1. **素材ドロップ**：
   - 通常ドロップ枠：一般的なエリア素材。
   - レアドロップ枠：モンスター固有の素材（魔獣の角、骨、魔法石など）。
   - ボスドロップ枠 (確率 100%)：次の施設解放キーアイテムや、大量の魔法石。

## 3. ボス討伐の仕様

エリアの探索度が 100% に達すると挑戦可能になる「ボス」との戦闘における固有の仕様です。

### ① パーティ編成と出撃

- ダンジョンリストに表示される「ボスと対決」ボタンから挑戦可能です。
- ボス戦には、村の中から**最大4人**を選択して派遣できます。

### ② 戦闘の進行

- ボス戦は通常の戦闘とは異なり、長期戦を想定した専用のUIで進行します。
- 画面上にはボスのHPゲージが表示され、現在の進行状況がリアルタイム（または1時間ステップごと）に視覚化されます。

### ③ ボスの自然回復

- ボスは強大な生命力を持っており、**時間の経過とともに体力が少しずつ回復**します。
- プレイヤーは、回復量を上回るダメージを継続的に与えられるよう、適切なメンバー選定と育成を行う必要があります。

## 4. 回復薬の自動使用システム [NEW]

村人は派遣前に倉庫の回復薬を自動的に所持してフィールドへ向かいます。戦闘や探索中にHPが低下すると自動で使用し、帰還時に余った分を倉庫へ返却します。

### 配布ルール

- 自動派遣時、倉庫に回復薬がある場合は **最大2個** まで自動的に村人に持たせます。
- 配布の優先順位は、効果量の高い薬品順に判定されます：
  1. `elixir` (エリクサー: HP完全に回復)
  2. `high_potion` (上級回復薬: HPを400回復)
  3. `mid_potion` (中級回復薬: HPを150回復)
  4. `potion` (回復薬: HPを50回復)
- 倉庫から消費分が差し引かれ、村人の `potionCount` および `potionItemId` にセットされます。

### 自動使用条件

以下のいずれかのタイミングで、村人のHPが最大HPの **50% 以下** かつ `potionCount > 0` の場合に自動使用されます：

| タイミング                   | 詳細                           |
| :--------------------------- | :----------------------------- |
| 探索行動前                   | 毎時間の採取・討伐処理前に判定 |
| 通常戦闘中（各ターン開始時） | ダメージを受ける前に判定       |
| ボス戦中（各ターン開始時）   | ボス戦の攻撃ラウンドごとに判定 |

- 1回の使用による回復量は、所持している回復薬の種類に基づきます（最大HPを超えない）。
- 使用のたびにゲームログに「[村人名] は回復薬を使用してHPを [回復量] 回復した」などのメッセージが記録されます。

### 返却ルール

- 村人が帰還を完了した際（`traveling_back` → `idle` or `resting`）、残っている `potionCount` をすべてそれぞれのアイテム種別に応じて倉庫（`inventory`）へ返却します。
- 帰還ログに「[村人名] が未使用の [回復薬名] N個を倉庫に返却しました」のメッセージが含まれます。

### UIへの反映

- 村人一覧の各カード右上に「🧪 ×N」バッジで所持数を表示します（0個の場合は非表示）。バッジをホバーした際や詳細画面には、所持している回復薬の具体的な名称が表示されます。
