# 施設＆クラフト設計書：加工と施設成長

村の施設（宿屋、工房、鍛冶場、錬金工房、交易所）の機能、アップグレードに必要な素材と効果、および素材を他のアイテムに加工するクラフトシステムの設計です。

---

## 1. 施設一覧とアップグレード仕様

各施設は、倉庫にある素材とゴールドを消費してアップグレード（レベルアップ）させることができます。

### ① 宿屋 (Inn)
村人の拠点。レベルを上げることで、雇える村人の最大数が増加し、活動していない村人のHP回復速度が向上します。
- **基本機能**：村人を宿屋に配置すると、1日経過時にHPが自動回復します。
- **アップグレード効果**：

| レベル | 最大村人枠 | 1日あたりのHP回復量 | 必要アップグレード素材・コスト |
| :---: | :---: | :---: | :--- |
| **Lv 1** | 3人 | 最大HPの 30% | 初期状態 |
| **Lv 2** | 4人 | 最大HPの 50% | 木材 ×20, 食料 ×50, 500 G |
| **Lv 3** | 6人 | 最大HPの 70% | 頑丈な木板 ×15, 干し肉 ×30, 1500 G |
| **Lv 4** | 8人 | 最大HPの 100% | 建築用石材 ×20, 鉄インゴット ×10, 5000 G |

---

### ② 工房 (Workshop)
一次素材である木材や石材を、高度な建築やアップグレードに必要な中間素材へと加工する施設です。
- **アップグレード効果**：レベルに応じて加工の同時実行数が増加し、上位レシピが解放されます。

| レベル | 同時加工枠 | 解放される加工レシピ | 必要アップグレード素材・コスト |
| :---: | :---: | :--- | :--- |
| **Lv 1** | 1枠 | 頑丈な木板, 建築用石材 | 初期状態 |
| **Lv 2** | 2枠 | 補強木材, 装飾石材 | 木材 ×30, 石材 ×30, 800 G |
| **Lv 3** | 3枠 | 高級建築部材 | 頑丈な木板 ×20, 鉄インゴット ×5, 2500 G |

---

### ③ 鍛冶場 (Forge)
鉱石から金属インゴットを鋳造し、村人が討伐（戦闘）の際に装備する「武器」や「防具」をクラフト・強化する施設です。
- **アップグレード効果**：装備のクラフト・強化レシピが解放され、装備の最大強化値が上昇します。

| レベル | 解放レシピ | 装備の最大強化値 | 必要アップグレード素材・コスト |
| :---: | :---: | :---: | :--- |
| **Lv 1** | 鉄インゴット, 鉄の剣, 鉄の鎧 | +3 | 木材 ×15, 鉄鉱石 ×20, 1000 G （Tier 1ボス撃破後） |
| **Lv 2** | 鋼鉄インゴット, 鋼鉄の剣, 鋼鉄の鎧 | +6 | 鉄インゴット ×15, 石炭 ×30, 3000 G |
| **Lv 3** | 魔鉄インゴット, 魔法の剣, 魔力外套 | +9 | 鋼鉄インゴット ×10, 魔法石 ×15, 8000 G |

---

### ④ 錬金工房 (Alchemy Lab)
薬草や魔法石を組み合わせ、戦闘や採取の効率を高める「薬品」や「魔導書」をクラフトする施設です。
- **アップグレード効果**：上位の回復薬や、一時的に能力値を上昇させるバフ薬品のレシピが解放されます。

| レベル | 解放レシピ | 必要アップグレード素材・コスト |
| :---: | :---: | :--- |
| **Lv 1** | 回復薬(小), 魔力薬(小) | 薬草 ×30, 魔法石 ×10, 1200 G （Tier 2ボス撃破後） |
| **Lv 2** | 回復薬(大), 活力の丸薬（物理攻撃力UP） | 薬草 ×50, 魔法石 ×25, 3500 G |
| **Lv 3** | 万能薬, 賢者の秘薬（魔力・魔法防御UP） | 高級薬草 ×20, 輝く魔法石 ×10, 9000 G |

---

### ⑤ 交易所 (Trading Post)
集めた素材を売却してゴールド（G）に換金したり、不足している素材を購入する施設です。
- **自動取引機能**：プレイヤーは「食料が50個未満になったら自動で20個購入する」「木材が100個を超えたら超えた分を自動売却する」といったルールを設定できます。
- **アップグレード効果**：取引手数料が減少し、一度に取引できる最大量が増加します。

| レベル | 取引手数料 | 自動取引設定枠 | 必要アップグレード素材・コスト |
| :---: | :---: | :---: | :--- |
| **Lv 1** | 20% | 2枠 | 頑丈な木板 ×10, 1500 G （Tier 3ボス撃破後） |
| **Lv 2** | 10% | 4枠 | 鉄インゴット ×10, 4000 G |
| **Lv 3** | 5% | 8枠 | 鋼鉄インゴット ×10, 交易所収益 10000G 達成 |

## 2. 加工（クラフト）システム

クラフトは、倉庫にある素材を消費して新しいアイテムを作り出すシステムです。

### クラフトの実行プロセス
1. **職人（ジョブ）の配置**：
   - 施設に村人（特に「職人」の職業）を配置してクラフトを行います。
   - 配置された村人の `DEX（器用さ）` に応じて、クラフト時間が短縮され、**大成功（Great Success）** の確率が上がります。
2. **大成功の効果**：
   - 消費素材や装備であれば、通常よりも高い性能（例：鉄の剣+1 など）で完成します。
   - 消費アイテム（薬や中間素材）の場合、完成数が 1.5倍〜2倍 になります。

### クラフトレシピデータ（JSON構造例）

Godotでのデータ管理用に、以下のような構造でレシピを定義します。

```json
{
  "craft_recipes": {
    "iron_ingot": {
      "facility": "forge",
      "required_level": 1,
      "materials": [
        {"item_id": "iron_ore", "amount": 3},
        {"item_id": "coal", "amount": 1}
      ],
      "result": {"item_id": "iron_ingot", "amount": 1},
      "work_required": 50
    },
    "iron_sword": {
      "facility": "forge",
      "required_level": 1,
      "materials": [
        {"item_id": "iron_ingot", "amount": 2},
        {"item_id": "wood_plank", "amount": 1}
      ],
      "result": {"item_id": "iron_sword", "amount": 1},
      "work_required": 100
    },
    "potion_small": {
      "facility": "alchemy_lab",
      "required_level": 1,
      "materials": [
        {"item_id": "herb", "amount": 3}
      ],
      "result": {"item_id": "potion_small", "amount": 1},
      "work_required": 30
    }
  }
}
```
*※ `work_required` は、村人が1日に提供する作業量（`DEX` や職業適性に基づく値）によって消化され、0になると完成します。*
