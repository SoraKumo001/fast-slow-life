# 職業（ジョブ）特性・仕様まとめ

ゲーム内の村人が就くことができる各職業（ジョブ）の特性、ステータス補正、転職条件、およびAI挙動への影響をまとめたリファレンスドキュメントです。

---

## 1. 職業システムの共通仕様

### 転職とコスト

- **新規転職コスト**: 各職業ごとに設定された基本コストに、ソウルアップグレードの「値切り上手」による割引（レベルごとに10%割引、最大50%割引）を適用した額のゴールドを消費します。
- **復職コスト（0 G）**: すでに一度でも転職したことのある職業（村人の `jobHistory` に含まれる職業）へ戻る際は、転職コストは常に **0 G** になります。
- **転職条件**:
  - **必要レベル**: キャラクターの現在レベルが指定値以上である必要があります。
  - **前提職業**: 上級職への転職には、前提となる特定の職業を習得（`jobHistory` に含まれる）している必要があります。

### ステータス再計算の仕組み

転職時、村人のステータスは以下の計算式に基づいて再計算されます。これにより、職業特有の能力値補正が即座に反映されます。

- **ベースステータス** = `10 + (ソウルアップグレード「頑強な肉体」のレベル × 2)`
- **レベルボーナス** = `(キャラクターレベル - 1) × 1.5`
- **各ステータス値** = `Math.floor((ベースステータス + レベルボーナス) × 職業ごとのステータス補正値)`
- **最大HP** = `Math.floor((100 + (キャラクターレベル - 1) × 10) × 職業ごとのVIT補正値)`

## 3. 各職業の固有特性とスキル

### 1. 無職

- **説明**: 初期状態。特に目立った特徴や補正はありません。
- **採取適性**: なし
- **自動派遣優先カテゴリ**: なし
- **特殊効果**: なし

### 2. 農民 (Farmer)

- **説明**: 食料生産のスペシャリスト。村の飢饉を防ぐための最重要ジョブ。
- **採取適性**: 食料 (`food`): **2.0倍**
- **自動派遣優先カテゴリ**: `food`
- **特殊効果**: なし

### 3. 鉱夫 (Miner)

- **説明**: 鉱石と石材の採掘のスペシャリスト。施設強化や装備クラフトの素材集めに適しています。
- **採取適性**: 鉱石 (`ore`): **2.0倍**、素材 (`material` ※石材・木材等): **1.5倍**
- **自動派遣優先カテゴリ**: `ore`, `material`
- **特殊効果**: なし

### 4. 薬師 (Pharmacist)

- **説明**: 薬草の採取効率が非常に高く、ポーション調合用の魔法石採取もこなします。
- **採取適性**: 薬草 (`herb`): **2.0倍**、魔法石 (`mana_stone`): **1.2倍**
- **自動派遣優先カテゴリ**: `herb`, `mana_stone`
- **特殊効果**: なし

### 5. 猟師 (Hunter)

- **説明**: 食料の採取効率が良く、討伐による戦利品（毛皮・骨など）のドロップ率を向上させます。
- **採取適性**: 食料 (`food`): **1.5倍**
- **自動派遣優先カテゴリ**: `food`, `material`
- **特殊効果**:
  - **ドロップ率ボーナス**: モンスター討伐時のアイテムドロップ抽選確率が **1.5倍** になります。

### 6. 戦士 (Warrior)

- **説明**: 物理戦闘のスペシャリスト。高いSTRとVITにより、ダンジョンの探索や強敵・ボスの討伐において主要な盾・アタッカーとなります。
- **採取適性**: なし
- **自動派遣優先カテゴリ**: `material`
- **特殊効果**:
  - **物理戦闘ボーナス**: 物理攻撃の与ダメージが **1.3倍 (30%増加)** になります。

### 7. 魔術師 (Mage)

- **説明**: 魔法戦闘と魔法石採取のスペシャリスト。極めて高いINTを持ち、魔法系の採取（薬草・魔法石）の効率を高めます。
- **採取適性**: 魔法石 (`mana_stone`): **2.0倍**
- **自動派遣優先カテゴリ**: `mana_stone`
- **特殊効果**: なし

### 8. 僧侶 (Priest)

- **説明**: 高い耐久力と回復スキルの使い手。探索での生存率を底上げし、薬草の採取もそつなくこなします。
- **採取適性**: 薬草 (`herb`): **1.3倍**
- **自動派遣優先カテゴリ**: `herb`
- **特殊効果**: なし

### 9. 職人 (Crafter)

- **説明**: 施設での生産活動のスペシャリスト。クラフトの作業時間を短縮し、大成功率を高めます。
- **採取適性**: 鉱石 (`ore`): **1.2倍**
- **自動派遣優先カテゴリ**: `ore`, `material`
- **特殊効果**:
  - **クラフト時間短縮**: クラフト作業を担当した際、必要所要時間が **20%短縮 (0.8倍)** されます。
  - **大成功率ボーナス**: クラフト時の大成功確率（成果物2倍）の算出において、職人のDEX補正(1.4)が適用され、確率が一律 **12%**（基本5% + 7%）に上昇します。

## 4. AI挙動（自動派遣）における影響

村人の「自動派遣」がオンになっている場合、職業はキャラクターの行き先と行動方針の決定に大きな影響を与えます。

### ① 不足アイテムの優先選択（優先カテゴリ）

倉庫の在庫がプレイヤーの「目標設定数」を下回っているアイテムがある場合、AIはそのアイテムを優先して採取（または討伐）しに行きます。その際、キャラクターの職業が持つ **「優先カテゴリ」** に含まれるアイテムから優先的にターゲットをソートします。

- （例：食料と鉱石が同時に不足している場合、農民は `food` である食料を、鉱夫は `ore` である鉱石を最優先で採取しに行きます）

### ② 採取スコアリングの補正

派遣先での自動ターゲット選択時、各採取ポイントに対する「適性スコア」が以下の計算式で算出されます。
職業ごとの **「採取適性」** と **「関連ステータス」** がスコアに二重に乗算されるため、適性の高い職業ほどそのアイテムを自発的に採取しやすくなります。

- **採取スコア計算式**:
  $$\text{Score} = \frac{1}{\text{難易度}} \times \text{採取適性倍率} \times \text{関連ステータス値} \times (1.0 + \text{AGI} \times 0.01) \times \text{スタミナ/空腹補正} \times \text{各種ペナルティ}$$
- **関連ステータス値の算出方法**:
  - `food` / `ore` / `material` カテゴリ: `STR × 0.7 + DEX × 0.3`
  - `herb` / `mana_stone` カテゴリ: `INT × 0.7 + DEX × 0.3`
- **採取の進行速度**:
  - 1時間あたりの採取ゲージ上昇量は `(DEX × 0.8 + 10) / 採取難易度` で計算されるため、DEXの高い「薬師」や「職人」は採取を早く終わらせることができます。
