# 画面構成・UI設計書

本作「Fast Slow Life」は、テキストおよびシンプルなUI（React & Tailwind CSS等）をベースとした画面構成を採用します。プレイヤーが村の状況、村人の活動、ダンジョンの探索状況を俯瞰しつつ、直感的に操作できるレイアウトを定義します。

---

## 1. 画面全体のレイアウト

画面は大きく分けて「基本情報エリア（上部）」「メインエリア（中部）」「ログエリア（下部）」の3つのセクションで構成されます。

## 2. 各エリアの仕様詳細

### 2.1. 基本情報エリア（画面上部）

ゲーム全体のステータス表示および時間進行のコントロールを行います。**P1 にて 2 段構成に再編**:

**Tier 1 (上段)**: アイデンティティ

- ロゴ "FAST SLOW LIFE" + "Web Edition"
- 時間 / Tier バッジ
- (保有時のみ) ソウルアップグレードバッジ
- 期限表示

**Tier 2 (下段)**: ステータス + 操作

- ゴールド表示 (破産時は赤字 + 警告)
- ソウルポイント (SP) 表示
- 操作ボタン群: 一時停止 / +24h / 速度 (1x・3x・10x) / ログ履歴 / リザルト / ヘルプ / 設定 / 転生

- **表示項目**:
  - **所持金**: 現在のゴールド。
  - **食料**: 現在の食料備蓄量。括弧内に現在の1日の消費予測（例: `-12/日`）を併記し、残り何日持つかの目安が直感的に伝わるようにします。**P0-3 にて < 2 日でパルス点滅警告を追加**。
  - **経過日時**: 現在の経過日数と時間（例: `Day 12 08:00`）— **Header にのみ表示**。StatusBar との重複を解消 (P0-2)。
- **コントロールボタン**:
  - `[ || ]` **一時停止**: 時間の自動進行を停止します。
  - `[  > ]` **通常再生 (1倍速)**: リアルタイム1秒につきゲーム内時間が1時間進みます。
  - `[ >> ]` **倍速進行 (3倍速/10倍速)**: リアルタイム1秒につきゲーム内時間が3時間/10時間進みます。
  - `[ +24h ]` **1日スキップ**: 24時間分を一瞬で裏側処理し、翌日の同刻まで進行させます（一時停止中のみ押下可能）。

### 2.1.1. ステータスバー (P1-2 にて 2 段化)

Header の直下に配置されるクイックステータス。

**Row 1**: 村人 / 平均Lv / 食料 / 進行度
**Row 2**: 地域 / 期限 / ログ (最新 3 件)

4 カラム メインの視認性を損なわないよう、太さ 2 行分。

### 2.2. メインエリア（画面中部）

本作の主要な情報源となる4つのリストを横並びに配置します。各リストの項目は**最小限の情報**にとどめ、クリックすることで詳細情報を表示するモーダル、あるいはサイドパネルが開く仕組みにします。

**P1-3 にて幅比を [1 : 1.2 : 1.2 : 1.4] に変更** (Inventory 狭め、Dungeon 広め)。情報量に応じた面積配分。

#### ① 素材・アイテムリスト

村の倉庫にあるアイテムの一覧です。

- **最小限表示（リスト上）**:
  - アイテム名、所持数
- **詳細表示（クリック時）**:
  - アイテムのアイコン、説明文、カテゴリ（一次素材/中間素材/装備品など）。
  - 売却価格（交易所での取引値）。
  - 用途（このアイテムを使用するクラフトレシピの一覧）。
  - 装備品の場合は、装備可能な職業、補正ステータス。

#### ② キャラクターリスト

村人たちの一覧です。

- **最小限表示（リスト上）**:
  - 名前、職業、HPバー、現在状態の簡易ステータス（「討伐中」「採取中」「クラフト中」「休息中」「死亡」など）。
- **詳細表示（クリック時）**:
  - **詳細ステータス**: HP、スタミナ、攻撃力、防御力、素早さ、知力、運。
  - **装備スロット**: 武器、防具、アクセサリー。タップで装備変更画面へ。
  - **自動行動の指示方針設定**:
    - 「採取」（どのダンジョン/エリアに行くかの方針選択）
    - 「討伐」（どのダンジョン/エリアに行くかの方針選択）
    - 「クラフト指示」（施設への割り当て）
    - 「休息」（宿屋でのHP/スタミナ回復）
  - **転職**: ゴールドを消費して他の職業にチェンジする画面への遷移。

#### ③ 施設リスト

村に建設・存在する施設の一覧です。

- **最小限表示（リスト上）**:
  - 施設名、レベル、現在の稼働状態（「アイドル」「クラフト進行中 (残り3時間)」「アップグレード中 (残り8時間)」など）。
- **詳細表示（クリック時）**:
  - 施設の概要、現在のレベルと効果（例: 鍛冶屋Lv.1「装備クラフト可能」、宿屋Lv.1「1時間あたりHP10回復」）。
  - **アップグレードボタン**: 次のレベルへのアップグレードコスト（ゴールドおよび素材）と所要時間の表示。
  - **クラフト・生産キュー**:
    - クラフト可能なアイテムレシピの一覧。
    - レシピを選択し、村人を割り当ててクラフトを開始する指示（時間経過によって進行）。

#### ④ フィールド・ダンジョンリスト

探索・攻略可能なエリアの一覧です。

- **最小限表示（リスト上）**:
  - エリア名、難易度（★マーク等）、現在の探索率（％）。
- **詳細表示（クリック時）**:
  - エリアの概要と背景。
  - **出現モンスター・採取素材のリスト**: 出現する可能性のある敵と獲得できる素材の一覧。
  - **現在派遣中の村人一覧**: そのエリアに「採取」または「討伐」で指示を出している村人のリスト。
  - **ボス挑戦ボタン**: 探索率が100%に達している場合、目標ボスへの挑戦を開始するボタン。

### 2.3. ログエリア

ゲーム内で発生したイベントを最小限のスペースで通知するエリアです。

- **実装 (P0〜)**: `StatusBar` (画面上部) の 2 段目に統合。専用行は作らず、KPI 表示と並列。
- **表示方式**:
  - 最新 3 件のログを `|` 区切りで結合表示。
  - クリックで `LogHistoryWindow` モーダルが開く (従来の動作と互換)。
  - ホバーで `title` 属性に各ログの全文 + タイムスタンプを表示。
- **カラーリングによる重要度識別**:
  - 一般ログ（採取・通常の戦闘）：ホワイト / 淡いグレー
  - 重要ログ（レベルアップ、クラフト完了）：グリーン / ブルー
  - 警告ログ（食料不足、村人の瀕死/死亡、ボスの出現、期限間近）：イエロー / レッド
- **履歴表示機能**:
  - ヘッダーの「ログ履歴」ボタン、または本ログ表示エリアをクリックで `LogHistoryWindow` モーダル (DraggableWindow) が表示され、フィルタリング可能なログ一覧が確認できる。

### 2.4. キーボードショートカット (P1〜)

ユーザー要望に基づき、以下のショートカットがグローバルで利用可能:

| キー            | 動作                                      |
| --------------- | ----------------------------------------- |
| `Space`         | 一時停止 / 再開                           |
| `1` / `2` / `3` | 速度切替 (1x / 3x / 10x)                  |
| `D`             | 1 日スキップ (一時停止中のみ)             |
| `R`             | リザルトモーダル表示                      |
| `L`             | ログ履歴モーダル表示                      |
| `?`             | 遊び方ヘルプモーダル表示                  |
| `Esc`           | 最前面のモーダルを閉じる (Modal 標準動作) |

- 入力欄フォーカス中 (`input` / `textarea` / `select` / `contenteditable`) は発火しない
- `Ctrl` / `Meta` / `Alt` 修飾キー押下時は発火しない
- ゲームオーバー時は一時停止と `?` のみ有効

### 2.5. アクティブボスバー (P2〜)

ボス戦が進行中、画面最上部に専用の固定バーを表示。DungeonPanel 内の表示より優先。

- **表示内容**:
  - ボス名 ("VS <name>")
  - HP バー + 現在値 / 最大値
  - 攻撃中アタッカーの頭文字アバター
  - 「撤退」ボタン
- **表示条件**: `activeBoss` が `null` でない間のみ表示
- **配置**: Header → StatusBar → ActiveBossBar → メイン 4 カラム の順

---

### 2.6. 設定ドロワー (P2〜)

画面右からスライドインするドロワー。ユーザー設定 (LocalStorage 永続化) を変更可能。

- **永続化キー**: `fsl-preferences`
- **設定項目**:
  - トースト通知 ON / OFF
  - トースト表示密度 (標準 / コンパクト)
  - アニメーション速度 (0.25x 〜 4x)
  - 高密度モード (余白を詰める)
  - ログ履歴保持件数 (10〜500)
  - 初回チュートリアル表示
- **起動**: ヘッダーの歯車アイコン

## 3. 追加モーダル・画面

### ボス戦モーダル

ダンジョンの探索度が100%に達したとき、ボスに挑戦するための村人選択と戦闘状況を表示するモーダル。

### 転生ショップ

ゲームオーバー時または手動リセット時に表示される「ソウルショップ」。獲得したソウルポイントを消費して永続バフを購入する画面。

### 交易馬車モーダル

交易所からアクセス可能な交易管理画面。馬車の派遣（輸出・輸入）、自動交易のON/OFF、街の投資、友好度や特産品の確認が行える。

### 結果画面（Result Screen）

ゲームオーバー時に表示される結果画面。プレイ時間、討伐数、クラフト数などの統計情報と獲得ソウルポイントを表示する。
