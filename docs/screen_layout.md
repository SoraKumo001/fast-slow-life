# 画面構成・UI設計書

本作「Fast Slow Life」は、テキストおよびシンプルなUI（Godotの Control ノード等）をベースとした画面構成を採用します。プレイヤーが村の状況、村人の活動、ダンジョンの探索状況を俯瞰しつつ、直感的に操作できるレイアウトを定義します。

---

## 1. 画面全体のレイアウト

画面は大きく分けて「基本情報エリア（上部）」「メインエリア（中部）」「ログエリア（下部）」の3つのセクションで構成されます。

```

|  [基本情報] 所持金: 1,200G | 食料: 450 (-12/日) | Day 12 08:00 [||] [>] [>>]  |
+-----------------------------------------------------------------------+
|                                                                       |
|  +----------------+ +----------------+ +----------------+ +--------+ |
|  | 素材・アイテム | |  キャラクター  | |      施設      | |エリア  | |
|  |                | |                | |                | |ダンジョ| |
|  | ・銅鉱石  x12  | | ・アリス (戦士)| | ・鍛冶屋  Lv.1 | | ・廃鉱山 | |
|  | ・薬草    x8   | |   [HP] ======= | |   [クラフト中] | |   [50%]  | |
```

## 2. 各エリアの仕様詳細

### 2.1. 基本情報エリア（画面上部）
ゲーム全体のステータス表示および時間進行のコントロールを行います。

- **表示項目**:
  - **所持金**: 現在のゴールド。
  - **食料**: 現在の食料備蓄量。括弧内に現在の1日の消費予測（例: `-12/日`）を併記し、残り何日持つかの目安が直感的に伝わるようにします。
  - **経過日時**: 現在の経過日数と時間（例: `Day 12 08:00`）。
- **コントロールボタン**:
  - `[ || ]` **一時停止**: 時間の自動進行を停止します。
  - `[  > ]` **通常再生 (1倍速)**: リアルタイム1秒につきゲーム内時間が1時間進みます。
  - `[ >> ]` **倍速進行 (2倍速/4倍速)**: リアルタイム1秒につきゲーム内時間が2時間/4時間進みます。
  - `[ +24h ]` **1日スキップ**: 24時間分を一瞬で裏側処理し、翌日の同刻まで進行させます（一時停止中のみ押下可能）。

---

### 2.2. メインエリア（画面中部）
本作の主要な情報源となる4つのリストを横並びに配置します。各リストの項目は**最小限の情報**にとどめ、クリックすることで詳細情報を表示するモーダル、あるいはサイドパネルが開く仕組みにします。

#### ① 素材・アイテムリスト
村の倉庫にあるアイテムの一覧です。
- **最小限表示（リスト上）**:
  - アイテム名、所持数
- **詳細表示（クリック時）**:
  - アイテムのアイコン、説明文、カテゴリ（一次素材/中間素材/装備品など）。
  - 売却価格（交易所での取引値）。
  - 用途（このアイテムを使用するクラフトレシピの一覧）。
  - 装備品の場合は、装備可能な職業、補正ステータス。

#### ② キャラクターリスト
村人たちの一覧です。
- **最小限表示（リスト上）**:
  - 名前、職業、HPバー、現在状態の簡易ステータス（「討伐中」「採取中」「クラフト中」「休息中」「死亡」など）。
- **詳細表示（クリック時）**:
  - **詳細ステータス**: HP、スタミナ、攻撃力、防御力、素早さ、知力、運。
  - **装備スロット**: 武器、防具、アクセサリー。タップで装備変更画面へ。
  - **自動行動の指示方針設定**:
    - 「採取」（どのダンジョン/エリアに行くかの方針選択）
    - 「討伐」（どのダンジョン/エリアに行くかの方針選択）
    - 「クラフト指示」（施設への割り当て）
    - 「休息」（宿屋でのHP/スタミナ回復）
  - **転職**: ゴールドを消費して他の職業にチェンジする画面への遷移。

#### ③ 施設リスト
村に建設・存在する施設の一覧です。
- **最小限表示（リスト上）**:
  - 施設名、レベル、現在の稼働状態（「アイドル」「クラフト進行中 (残り3時間)」「アップグレード中 (残り8時間)」など）。
- **詳細表示（クリック時）**:
  - 施設の概要、現在のレベルと効果（例: 鍛冶屋Lv.1「装備クラフト可能」、宿屋Lv.1「1時間あたりHP10回復」）。
  - **アップグレードボタン**: 次のレベルへのアップグレードコスト（ゴールドおよび素材）と所要時間の表示。
  - **クラフト・生産キュー**:
    - クラフト可能なアイテムレシピの一覧。
    - レシピを選択し、村人を割り当ててクラフトを開始する指示（時間経過によって進行）。

#### ④ フィールド・ダンジョンリスト
探索・攻略可能なエリアの一覧です。
- **最小限表示（リスト上）**:
  - エリア名、難易度（★マーク等）、現在の探索率（％）。
- **詳細表示（クリック時）**:
  - エリアの概要と背景。
  - **出現モンスター・採取素材のリスト**: 出現する可能性のある敵と獲得できる素材の一覧。
  - **現在派遣中の村人一覧**: そのエリアに「採取」または「討伐」で指示を出している村人のリスト。
  - **ボス挑戦ボタン**: 探索率が100%に達している場合、目標ボスへの挑戦を開始するボタン。

### 2.3. ログエリア（画面下部）
ゲーム内で発生したイベントを最小限のスペースで通知するエリアです。

- **表示方式**:
  - 改行を行わず、1行のテキスト領域として配置します。
  - 最新のログが右側から流れるように追加され、区切り文字（`|` や `*` など）で連結されていきます。
  - `[10:00] アリスがゴブリンを討伐 (経験値+10) | [10:15] ボブが薬草を採取 | [11:00] 鍛冶屋で「鉄の剣」が完成`
- **カラーリングによる重要度識別**:
  - 一般ログ（採取・通常の戦闘）：ホワイト / 淡いグレー
  - 重要ログ（レベルアップ、クラフト完了）：グリーン
  - 警告ログ（食料不足、村人の瀕死/死亡、ボスの出現、期限間近）：イエロー / レッド
- **履歴表示機能**:
  - ログエリアの右端に `[ログ履歴]` ボタンを設置し、クリックすると過去ログ（最大200件程度）をスクロール可能なリスト形式で閲覧できるポップアップウィンドウを表示します。
